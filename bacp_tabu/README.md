## Estado de Avance — Oscar Mauricio Ruiz Jimenez

---

## Compilación

```bash
make
```

Para limpiar los archivos de compilación:

```bash
make clean
```

## Ejecución

```bash
./bacp <archivo_instancia.txt>
```

**Ejemplos:**
```bash
./bacp bacp8.txt
./bacp utfsm7313.txt
```
