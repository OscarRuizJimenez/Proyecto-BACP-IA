## Compilación

```bash
make
```

Para limpiar los archivos de compilación:

```bash
make clean
```

## Ejecución

```bash
./bacp instancias/<archivo_instancia.txt>
```

**Ejemplos:**
```bash
./bacp instancias/bacp8.txt
./bacp instancias/utfsm7313.txt
```
